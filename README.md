# 📝 Modern Todo App

A minimal, fast, and responsive task management application built to help keep track of daily routines and boost productivity. Designed with a clean, modern UI for the best user experience.

## ✨ Features

- **Task Management:** Easily add, edit, and mark tasks as completed.
- **Modern UI/UX:** Clean, minimalistic, and intuitive interface.
- **Lightning Fast:** Powered by Vite for instant server start and fast HMR.

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Standard CSS
- **Build Tool:** Vite
